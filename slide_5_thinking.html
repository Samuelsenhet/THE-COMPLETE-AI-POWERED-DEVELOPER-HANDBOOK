The Thinking Framework

Four Levels of Problem Solving

01

Logical What is?

Define the problem space, understand core requirements, and identify stakeholders.

02

Analytical How do I?

Break down components, identify dependencies, and map technical resources.

03

Computational How to fit logic?

Structure solution architecture, design system interactions, and plan data flows.

04

Procedural How do I excel?

Optimize implementation, refine code quality, and apply best practices.

Example Prompt
"Help me think through my social media analytics dashboard using all 4 levels"
